git is  a distributed  version controll system designed to take care of projects with speed and accuracy.starting wtih git init command wich initialize the empty repository in  /.git. repository is a hidden directory where repository operates or where git has been intialized to start version contolling your files.and directory is folder used for storing multiple files.
status is to see what the current state of our project.git add <file name>.txt adds the files to staging area.and status shows tracked untracked , nofiles to track .git log shows how many commits done till the moment.git commit to store our staged changes we use this command.after this commit you nowhere can use status .for that you need to use log command.git push is to push your file to the git hub repository.
